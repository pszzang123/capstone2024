## 📋개요

소비자가 PC뿐만 아니라 모바일에서 의류를 구매할 수 있는 서비스를 개발했습니다.

또한 판매자는 PC에서 판매할 의류를 등록할 수 있습니다.

<br/>

## 📋프로젝트 환경 설정 (개발환경)
- 프레임워크 : React, Flutter, Spring Boot
- DBMS : MySQL & Firebase
- UI 환경 : PC 및 모바일 기기 (Android, IOS)
